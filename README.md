# 📊 Trading Coach - 交易复盘系统

AI驱动的个人交易复盘工具，通过深度分析历史交易数据，帮助投资者系统性地总结交易经验、识别交易模式、发现优势与不足。

## ✨ 功能特性

- 📊 **交互式仪表盘**: 关键指标卡片、盈亏曲线、AI洞察
- 📋 **交易列表管理**: 多维度筛选、排序、导出
- 📅 **周期性报告**: 自定义时间范围的详细分析
- 🔍 **单笔交易详情**: 深度分析每笔交易
- 🤖 **AI智能洞察**: 自动识别问题和机会
- 📈 **可视化图表**: 交互式Plotly图表

## 🚀 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行Web应用

```bash
streamlit run app.py
```

应用会自动在浏览器中打开（默认：http://localhost:8501）

## 📷 功能预览

### 仪表盘
- 关键指标概览
- 累计盈亏曲线
- AI洞察和警告
- 市场分布分析

### 周期性报告
- 自定义时间范围
- 盈亏分析和分布
- 持仓时间统计
- 标的交易频率

### 交易列表
- 按市场/方向/标的筛选
- 完整交易记录
- CSV导出功能

## 📊 核心指标

系统自动计算以下指标：

- 总盈亏、胜率、盈亏比
- 最大单笔盈利/亏损
- 平均持仓时间
- 交易频率分析
- 费用占比分析

## ⚠️ 注意事项

- MVP版本使用简化的配对交易算法
- 复杂交易（分批建仓/减仓）可能配对不准确
- 建议定期备份交易数据
